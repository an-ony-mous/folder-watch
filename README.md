# folder-watch
対象フォルダを監視するサンプルプログラム

## 処理内容

・特定のフォルダの状態を監視するプログラムを作成してみました。
※簡単な動作確認しかしていませんが。。。(汗)

・単体テストは別途作成する予定です。

・コンソールアプリを起動すると「Setting.json」内のフォルダを監視し続けます。

・アプリケーションを終了する場合は「ESC」キーで終了します。

・起動時にサブフォルダを辞書登録して、そのフォルダのファイル数を監視しています。

・ESCキーにて最終のフォルダ内のファイル数をコンソール出力します。
